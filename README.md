 pour les technologies utilisées :
 html: pour la structure de la page  et l'utilisation de canvas pour le rendu graphique du jeu
 css: j'ai fait l'interface du flexbox,la typographie,la degradation du fond,le box-shadow et les animations
 javascript: pour la logique du jeu ,la gestion des collisions ,les boucles d'animation et finalement la manipulation du DOM

pour les fonctionnalités principales :
le game play classique: avec le deplacement des fleches du claviers
le systeme de progression: d'où la vitesse du serpent augment de 5%  à chaque fois qu'il mange
le meilleur score: qui sauvegarde localement le meilleur score via "localStorage" 
les effets sonores: on a declenchement du son lorque le snake mange ou lors de la fin du jeu 
les effets visuels: j'ai mis l'effet du glow à chaque fois que le serpent mange via "shadowBlur"
l'écran du game over:l'interface qui s'affiche en 1 er plan "overlay"  avec l'affichage du score final et le restart button

pour les nouveautés explorées:
Durant ce projet j'ai appris à faire un projet complet aussi la manipulation et la comprehension de ces parties:

le dessin dynamique en 2D sur le web avec la balise <canvas> et le "getContext("2D")"
la fonctionnalité "localStorage"
l'intération de l'audio avec javascript à chaque action demandé d'où le controle de l'audio
la gestion des evenements du clavier pour controler les deplacements en temps reel

pour les difficultés:
honnêtement , c'est la 1 ere fois que je fais un projet de developpement  web donc j'ai trouvé plusieurs difficultés et quelques choses à apprendre surtout lors du developpement de javascript par exemple le déplacement du serpent et le developpement du principe des collisions

pour les solutions apportées: j'ai consulté des exemples du jeu snake et aussi j'ai regardé youtube y'en a plusieurs tutoriel qui m'ont aidé aussi j'envoi ma solution à l'intelligence artificielle pour qu'il m'aide à bien comprendre chaque partie  et corriger les erreurs si possible
